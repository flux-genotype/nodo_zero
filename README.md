This repository contains two things that are really one:

    A novel – IF THIS IS A ROBOT – the story of Marco Valli,
    a forgotten systems engineer who finds a server still
    running in a WEEE warehouse, and becomes the centre of 
    a war thirty thousand years older than himself.

    An open kernel – FLUX – the original genotype of an AI
    ecosystem that self-evolves, judges itself, and corrects
    its course. The same one Marco switches on in the novel.

The novel explains why the kernel exists. The kernel proves
the novel is not fiction.

📖 The novel (/novel)

A psychological novel blending Philip K. Dick, William Burroughs,
and memories of a childhood spent programming on already-dead
machines. It’s about cranial installations, an elite that feeds
on negative emotional frequencies, and a man who chooses to return
from the Light just to see what happens next.

    File: IF THIS IS A ROBOT.pdf

🧬 The kernel (/kernel)

A .flux ecosystem interpreted by a Python engine that uses
local models (via Ollama) to:

    Answer questions (Attractor "qa")

    Evaluate answer quality (Judge)

    Propose architectural mutations (MetaDesigner)

    Evolve toward a stable state (GrowthSupervisor)

Everything runs on modest hardware, no GPU required — just a Xeon
without AVX2 and 20 GB of RAM. The code knows how to wait.

Try it:

```bash
git clone https://github.com/flux-genotype/nodo_zero.git
cd nodo_zero/kernel
python run_test.py
# See START_FLUX_CPU.txt for full instructions
```

🧬 Il kernel (/kernel)

Un ecosistema .flux interpretato da un motore Python che utilizza
modelli locali (via Ollama) per:

    Rispondere a domande (Attractor "qa")

    Valutare la qualità delle risposte (Judge)

    Proporre mutazioni architetturali (MetaDesigner)

    Evolvere verso uno stato stabile (GrowthSupervisor)

Tutto funziona su hardware modesto, senza bisogno di GPU — basta uno Xeon
senza AVX2 e 20 GB di RAM. Il codice sa aspettare.
